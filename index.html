[ 01 / About ]

Building interfaces an web page

I am a front-end web developer with hands-on experience in HTML5, CSS3, and JavaScript. My interest in design began in 2016 through self-study in Photoshop and visual design, and this interest has stayed with me ever since — through the Web Development Challenge path (ITIDA & Udacity), UI/UX and front-end development training with ITI, and recently the Web Designer program with NTI.

I’m into clean, responsive designs, paying attention to the fine details of the user interface, and creating interfaces that are easy to use. My practical projects — an e-commerce page, a real estate listings site, and a landscaping company website — reflect this interest: each project was built around a real idea, from the prototype to a working user interface.

I continue developing these foundations, project by project.
