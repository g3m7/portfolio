Building interfaces for the web

I started exploring design back in 2016 through self-study, and today I build professional web interfaces with HTML5, CSS3, and JavaScript — skills I developed hands-on through the Web Development Challenger Track (ITIDA & Udacity), UI/UX & Front-End training with ITI, and most recently the Web Designer program with NTI.
